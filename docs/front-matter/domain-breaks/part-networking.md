# Part 1: Networking Fundamentals

This section covers the foundational concepts that every network engineer must understand before moving to any specialisation. It begins with the OSI model as the organising framework, moves to physical topology and media, and establishes the vocabulary of collision domains, broadcast domains, and addressing that every subsequent section builds on.

By the end of this section, you will be able to describe how data moves from one device to another, identify the physical and logical structure of a network, and select appropriate media for a given link.

**Modules in this section:** NW-001 The OSI Model, NW-002 Network Topologies, NW-003 Ethernet Standards and Cabling
