# Part 3: Routing Protocols

This section covers how routers learn about remote networks and decide which path to take. It begins with the mechanics of routing itself, moves through static routing, and then covers the major dynamic routing protocols: distance-vector (RIP), link-state (OSPF), and path-vector (BGP). Advanced topics include multi-area OSPF, BGP policy, and route redistribution between protocols.

Routing knowledge is the core skill of network engineering. Every role in this compendium relies on the concepts covered here.
