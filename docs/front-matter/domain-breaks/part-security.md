# Part 6: Security

This section covers network security at the infrastructure level: access control lists for traffic filtering, firewall concepts and stateful inspection, VPN and IPsec for encrypted tunnels, AAA (authentication, authorisation, and accounting), encryption standards and PKI, and network segmentation using DMZ architecture. Security is not a separate discipline from network engineering; it is built into every design decision.

**Modules in this section:** SEC-001 Access Control Lists, SEC-002 Firewall Concepts, SEC-003 VPN and IPsec, SEC-004 AAA: TACACS+ and RADIUS, SEC-005 Encryption Standards and PKI, SEC-006 Network Segmentation and DMZ Design
