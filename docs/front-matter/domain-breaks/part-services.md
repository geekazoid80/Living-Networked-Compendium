# Part 5: Network Services

This section covers the infrastructure services that make IP networks usable: DNS for name resolution, DHCP for address assignment, NAT and PAT for address translation, NTP for time synchronisation, and SNMP/Syslog for monitoring and logging. These services run on virtually every network and are among the first things a junior engineer will need to configure and troubleshoot.

**Modules in this section:** SV-001 DNS, SV-002 DHCP, SV-003 NAT and PAT, SV-004 NTP and Time Synchronisation, SV-005 SNMP and Syslog
